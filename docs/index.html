Fonctionnalités

🎵

Crossfade

Transition fluide equal-power entre les morceaux. Durée configurable.

🎨

Skins .wsz

Chargez vos skins Winamp classiques. Le look que vous aimez, en haute définition.

📴

100 % Offline

Fonctionne sans internet. Vos fichiers, votre musique, point final.

💻

Multi-plateforme

Windows, macOS et Linux. Un seul lecteur partout.

📂

Playlists M3U

Créez, éditez et sauvegardez vos playlists au format standard.

🔓

Open Source

Code ouvert sur GitHub. Contribuez, forkez, personnalisez.
